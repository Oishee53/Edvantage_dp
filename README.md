# Edvantage - Your Virtual Classroom

Edvantage is a centralized Learning Management System (LMS) that enables instructors to create and share courses while allowing students to learn through structured materials such as videos, PDFs, quizzes, and live classes. The platform ensures quality through admin approval, supports progress tracking, and enhances learning with interactive features like discussion forums, Q&A, and AI-powered chatbots.

🚀 Key Features

🔐 Authentication & role-based access (Student, Instructor, Admin)
📚 Course creation, approval, and management
🛒 Course browsing, wishlist, cart, and enrollment
📊 Progress tracking (video completion + quiz performance)
❓ Lecture-based Q&A system
💬 Discussion forum with threads, replies, and reactions
🎥 Live class integration (video, audio, screen sharing, chat)
📝 Online written exams with screen & webcam monitoring
📂 Assignment submission and grading system
🤖 AI chatbots (platform assistant + document-based Q&A)
🧠 OCR-based exam & quiz generator
⭐ Course rating and filtering system
🎯 Hybrid course recommendation system
🎓 Certificate generation with verification

🛠️ Tech Stack
Frontend: Laravel (Blade), JavaScript
Backend: PHP
Database: MySQL
Version Control: Git, GitHub
IDE: VS Code
Video Hosting: Cloudinary
Streaming: Mux
